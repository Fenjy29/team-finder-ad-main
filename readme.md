# TeamFinder — инструкция для ревьюера

## Быстрый запуск (Docker Compose — рекомендуемый способ)

Требования: Docker и Docker Compose установлены.

```bash
# 1. Клонировать репозиторий
git clone <repo-url>
cd team-finder

# 2. Запустить всё одной командой
docker compose up --build
```

Приложение автоматически:
- поднимет PostgreSQL в контейнере
- применит миграции
- заполнит базу тестовыми данными
- запустит сервер Django на http://localhost:8000

### Остановка

```bash
docker compose down
```

Данные сохраняются в Docker volumes (`postgres_data`, `media_data`) и не теряются при перезапуске.

## Локальный запуск (без Docker)

Требования: Python 3.10+, PostgreSQL.

```bash
# 1. Создать и активировать виртуальное окружение
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Настроить .env (файл уже есть, при необходимости скорректируйте параметры БД)
# По умолчанию используется PostgreSQL на localhost:5432

# 4. Запустить PostgreSQL (если не запущен)
docker compose up db -d

# 5. Применить миграции и заполнить тестовыми данными
python manage.py migrate
python manage.py seed_data

# 6. Запустить сервер
python manage.py runserver
```

Приложение будет доступно по адресу http://localhost:8000.

## Страницы и функциональность

| URL | Описание |
|-----|----------|
| `/projects/list/` | Главная — список проектов (пагинация, фильтр по навыкам) |
| `/projects/<id>/` | Страница проекта |
| `/projects/create-project/` | Создание проекта |
| `/projects/<id>/edit/` | Редактирование проекта |
| `/users/list/` | Список участников |
| `/users/<id>/` | Профиль пользователя |
| `/users/register/` | Регистрация |
| `/users/login/` | Вход |
| `/users/edit-profile/` | Редактирование профиля |
| `/users/change-password/` | Смена пароля |
| `/admin/` | Административная панель |

## Вариант задания

Реализован **Вариант 3** — навыки проектов и фильтрация:
- На странице проекта отображается блок «Необходимые навыки»
- Владелец проекта может добавлять/удалять навыки без перезагрузки страницы
- Автодополнение при вводе навыка, возможность создать новый
- На главной странице работает фильтрация проектов по навыкам
- Активный фильтр подсвечивается, есть кнопка сброса
